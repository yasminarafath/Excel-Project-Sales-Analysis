The process of building an interactive Bike Purchase Analysis Dashboard using Excel! The highlights of data analysis project using excel! involves analyzing sales data using Microsoft Excel, focusing on various factors related to customers and their purchasing behavior. The project includes steps such as data cleaning and pre-processing, creating pivot tables and charts, and building an interactive sales analysis dashboard.

Key Insights from Bike Sales Data:

Customer Demographics: Insights into marital status,revenue,and occupation.

Purchase report: Analysis of purchase by revenue.

Visualizations: Various charts and an interactive dashboard.

The goal is to gain insights into understanding customer behavior and sales trends, which can inform business decisions and strategies.
